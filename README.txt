Проделанная работа по основной части задания:

1. Реализован вывод данных в таблицу из JSON-файла с использованием технологии AJAX.
2. Добавлена возможность сортировки образованных столбцов по алфавиту - в прямом и обратном порядке через метод sort
3. По щелчку на ячейку реализован вывод формы редактирования содежимого всей строки с помощью конструирования элемента формы и занесения туда данных из таблицы
4. Данные колонки About обрезаются после 2х строк в каждой ячейке благодаря технологии -webkit
