Выполнены следующие задания:

Основные цели: 
    • На основе имеющегося массива данных, создать таблицу шириной 50% от экрана;
    • Колонки в таблице: 
        ◦ Имя (firstName), 
        ◦ Фамилия (lastName), 
        ◦ Описание (about),
        ◦ Цвет глаз (eyeColor).
    • Данные колонки “about” должны отображаться в две строки, остальное обрезаем многоточием (...);
    • Каждая колонка должна иметь свойство сортировки;
    • При клике на строку в соседнем с таблицей DIV’е отобразить форму редактирования данных выбранной строки.

Дополнительные цели:
    • Добавить возможность показа/скрытия колонок;
    • В колонке “eyeColor” предоставлять данные в виде цвета, сохраняя возможность сортировки по значению.

Идеи по реализации этого задания
    • Создать несколько файлов HTML и связать их через PHP. При том, для каждого HTML будет либо свой JS файл, либо будет отдельно от одного общего создана функция вывода данных, где они будут выводиться с i = n по i < n+10 пользователя. С одной страницы на другую переход будет явным, это будут фактически разные страницы и в адресной строке будут разные строки.
    • Другой путь - снова AJAX. Необходимо будет создать три несколько разных HTML файлов и связать их через теги <a> с href="secondPage.HTML". Это - способ создания полноценного веб-приложения, при том динамического, размещенного на одной странице.
